# Overview

BlockVote is a secure, transparent, and tamper-proof online voting system built using blockchain technology. It eliminates traditional voting issues like fraud, manipulation, and lack of transparency by recording votes on an immutable decentralized ledger.

## Problem Statement

Traditional voting systems:

Lack transparency
Are prone to tampering and fraud
Require manual counting → delays & errors
Depend on centralized authorities

## Our Solution

Ensure secure and tamper-proof voting, Provide real-time vote tracking, Enable instant result calculation, Maintain immutable vote records, Preserve voter anonymity

## Tech Stack

Frontend: React 
Backend: java script, postman
Blockchain: javascript
